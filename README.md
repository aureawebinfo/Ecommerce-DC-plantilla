# Ecommerce Productos Típicos

Ecommerce desarrollado con el stack tecnológico de Áurea Web:

- **Frontend:** Next.js + React + Tailwind CSS
- **Backend:** Django + Django REST Framework  
- **Base de datos:** PostgreSQL
- **Patrones:** MVVM (Frontend) + DDD (Backend)

## Productos:
- Mamona
- Kumis
- Chicharrones  
- Masato
- Alfajores
